# primer-scout v0.1.0

Release date: 2026-02-18

Initial public release of `primer-scout`.

## Highlights

- Fast primer off-target scanning for FASTA references
- Mismatch-aware matching via `--max-mismatches`
- Reverse-complement scanning enabled by default
- IUPAC-aware sequence handling
- Three output modes:
  - hit-level rows
  - per-primer summary (`--summary`)
  - count-only (`--count-only`)
- TSV and NDJSON output support
- Deterministic benchmark pipeline and committed benchmark artifacts

## CLI Examples

```bash
primer-scout --primers primers.tsv --reference ref.fa --max-mismatches 1
primer-scout --primers primers.tsv --reference ref.fa --summary
primer-scout --primers primers.tsv --reference ref.fa --count-only
```

## Performance Evidence (local, reproducible)

Macro benchmark artifact: `benchmarks/RESULTS.md`
- Dataset: 5,000,000 bases, 128 primers, primer length 20, `k=1`
- Mean runtime: 1.214 s
- Throughput: 4.119 million-bases/s

Microbenchmark artifact: `benchmarks/CRITERION_RESULTS.md`
- Includes workload timings for `32/128 primers` at `k=0/1`
